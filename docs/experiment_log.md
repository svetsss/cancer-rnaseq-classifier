# Журнал экспериментов

## 2026-07-16 — первичный анализ UCI RNA-Seq

Цель запуска — загрузить исходный архив UCI, отделить целевую переменную от матрицы
признаков, проверить структуру данных и сохранить воспроизводимые артефакты этапа 1.

Команда:

```bash
python -m scripts.analyze_dataset
```

Окружение: Python 3.12.13, NumPy 1.26.4, pandas 3.0.3, Matplotlib 3.11.0.

Получены следующие характеристики:

- 801 объект и 20 531 числовой признак типа `float64`;
- 5 классов: BRCA — 300, COAD — 78, KIRC — 146, LUAD — 141, PRAD — 136;
- 0 пропусков и 0 бесконечных значений;
- 0 повторяющихся идентификаторов, имён признаков и строк признаков;
- целевая переменная сохранена отдельно под именем `cancer_type` и отсутствует в `X`.

SHA-256 загруженного архива:
`e6ea3628f9656efa2d7cf382bf5a403e2cc214df3d90312b1928f5650b43a559`.

Созданы `results/dataset_summary.json` и `figures/class_distribution.png`. Исходный
архив сохранён в `data/raw/` и исключён из Git.

Обычный запуск тестов не обращается к сети: загрузчик проверяется на малом архиве,
созданном во временном каталоге. Итог проверки: Ruff — без ошибок, mypy — без ошибок,
pytest — 6 тестов пройдено.

## 2026-07-16 — разделение данных и baseline-модели

Команда запуска:

```bash
python -m scripts.run_baselines
```

Датасет разделён функцией `train_test_split` с `test_size=0.2`, `random_state=42` и
`stratify=y`. Получено 640 объектов в train и 161 объект в test.

| Класс | Train | Test |
|---|---:|---:|
| BRCA | 240 | 60 |
| COAD | 62 | 16 |
| KIRC | 116 | 30 |
| LUAD | 113 | 28 |
| PRAD | 109 | 27 |

Cross-validation выполнена только на train-части: `StratifiedKFold(n_splits=5,
shuffle=True, random_state=42)`. Test set не передавался в preprocessing, обучение или
оценку и не использовался для выбора конфигураций.

Проверены две конфигурации на всех 20 531 признаках:

- E00: `DummyClassifier(strategy="most_frequent", random_state=42)`;
- E01: `StandardScaler` и `LogisticRegression(max_iter=3000, random_state=42)` внутри
  `Pipeline`.

Фактические результаты 5-fold cross-validation:

| ID | Macro F1, mean ± std | Accuracy, mean ± std | Precision macro, mean ± std | Recall macro, mean ± std | Fit mean, с | Score mean, с | Всего, с |
|---|---:|---:|---:|---:|---:|---:|---:|
| E00 | 0.109091 ± 0.000000 | 0.375000 ± 0.000000 | 0.075000 ± 0.000000 | 0.200000 ± 0.000000 | 0.1563 | 0.0027 | 0.83 |
| E01 | 0.995388 ± 0.006527 | 0.995313 ± 0.006250 | 0.993517 ± 0.009372 | 0.997500 ± 0.003333 | 0.3791 | 0.0709 | 2.29 |

DummyClassifier предсказывает только наиболее частый класс. Для precision остальных
классов используется определённое значение 0, поэтому ожидаемая ситуация не создаёт
предупреждения о неопределённой метрике. Logistic Regression завершилась без предупреждения
о сходимости. Константные признаки на этом этапе не удалялись, отбор признаков и настройка
гиперпараметров не выполнялись.

Созданы `results/split_summary.json` и `results/metrics.csv`. Файлы записываются через
временный файл и `Path.replace`; повторный запуск заменяет строки E00 и E01, не добавляя
дубликаты. Финальные test-метрики не вычислялись.
