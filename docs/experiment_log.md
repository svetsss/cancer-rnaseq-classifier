# Журнал экспериментов

## 2026-07-16 — первичный анализ UCI RNA-Seq

Цель запуска — загрузить исходный архив UCI, отделить целевую переменную от матрицы
признаков, проверить структуру данных и сохранить воспроизводимые артефакты этапа 1.

Команда:

```bash
python -m scripts.analyze_dataset
```

Окружение: Python 3.12.13, NumPy 1.26.4, pandas 3.0.3, Matplotlib 3.11.0.

Получены следующие характеристики:

- 801 объект и 20 531 числовой признак типа `float64`;
- 5 классов: BRCA — 300, COAD — 78, KIRC — 146, LUAD — 141, PRAD — 136;
- 0 пропусков и 0 бесконечных значений;
- 0 повторяющихся идентификаторов, имён признаков и строк признаков;
- целевая переменная сохранена отдельно под именем `cancer_type` и отсутствует в `X`.

SHA-256 загруженного архива:
`e6ea3628f9656efa2d7cf382bf5a403e2cc214df3d90312b1928f5650b43a559`.

Созданы `results/dataset_summary.json` и `figures/class_distribution.png`. Исходный
архив сохранён в `data/raw/` и исключён из Git.

Обычный запуск тестов не обращается к сети: загрузчик проверяется на малом архиве,
созданном во временном каталоге. Итог проверки: Ruff — без ошибок, mypy — без ошибок,
pytest — 6 тестов пройдено.
