# Ограничения

- Набор содержит 801 объект при 20 531 признаке, поэтому последующие эксперименты будут
  чувствительны к переобучению и способу валидации.
- Имена признаков условные (`gene_0`, `gene_1`, ...). По ним нельзя делать выводы о
  конкретных генах или биологических маркерах.
- В проекте используется один открытый датасет без внешней валидации.
- Данные и результаты относятся только к учебной задаче классификации пяти классов опухолей.
- Проект не проходил клиническую проверку и не предназначен для диагностики или работы с
  пациентами.
