# Область применения и границы эксперимента

- Набор содержит 801 объект при 20 531 признаке, поэтому последующие эксперименты будут
  чувствительны к переобучению и способу валидации.
- Имена признаков условные (`gene_0`, `gene_1`, ...), поэтому биологическая интерпретация
  требует отдельного mapping к идентификаторам генов.
- Текущий эксперимент построен на одном открытом датасете UCI.
- Целевая переменная содержит пять типов опухолевой ткани: BRCA, COAD, KIRC, LUAD и PRAD.
- Финальная точечная оценка получена на 161 объекте. Для accuracy 161/161 нижняя граница
  95% Wilson interval составляет около 0.977.
- Repeated cross-validation на train-части дал macro F1 `0.9980 ± 0.0045` при диапазоне
  `0.9873–1.0000`. Это проверяет чувствительность к разбиению, но не перенос между когортами.
- Streamlit принимает подготовленные признаки в шкале UCI с зафиксированными именами
  столбцов.
- Для новой когорты предусмотрены harmonization, проверка batch effects и внешняя оценка.

План честного закрытия внешней проверки описан в
[`external_validation_protocol.md`](external_validation_protocol.md).
